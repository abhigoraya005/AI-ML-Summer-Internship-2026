## Assignment Overview

The assignment explores the Titanic Disaster dataset and its relevance to Machine Learning and Data Science. Key topics covered include:

### Theoretical Questions

1. Lessons learned from the Titanic Disaster
2. Why distractions are dangerous
3. Importance of keeping stakeholders informed
4. Need for traceability in engineering and ML projects
5. Benefits of documentation and knowledge preservation

### Practical Exercises

#### Exercise 1: Linear Regression

* Predicting passenger fare using:

  * Age
  * Passenger Class (Pclass)
  * Number of Siblings/Spouses (SibSp)

#### Exercise 2: Logistic Regression

* Predicting passenger survival
* Classification problem using:

  * Age
  * Passenger Class
  * Gender

#### Exercise 3: Supervised vs Unsupervised Learning

* Logistic Regression (Supervised Learning)
* K-Means Clustering (Unsupervised Learning)
* Comparison of labelled and unlabelled learning approaches
