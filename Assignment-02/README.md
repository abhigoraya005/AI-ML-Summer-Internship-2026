# Assignment 2 – Titanic Dataset Analysis & Visualization Ethics

## Overview
This assignment focused on analyzing the Titanic Passenger Survival Dataset and understanding the importance of ethical data visualization.

## Objectives
- Study the Titanic Dataset
- Identify the most suitable chart for survival analysis
- Understand visualization ethics
- Analyze how poor chart design can mislead viewers

## Question 1
### Which chart best shows Titanic survival by gender and class?

### Selected Chart
**Grouped Bar Chart**

### Reason
A grouped bar chart effectively compares:
- Male vs Female passengers
- 1st, 2nd, and 3rd passenger classes
- Survival and non-survival counts

### Key Observations
- Female passengers had significantly higher survival rates.
- First-class passengers had the highest survival chances.
- Third-class male passengers had the lowest survival rate.
- Passenger class and gender strongly influenced survival.

## Question 2
### How can poor visualization design distort information?

### Common Issues
1. Misleading Scales
2. Excessive Colors
3. Cherry-Picked Data
4. Wrong Chart Selection

### Importance
Good visualizations should:
- Use honest scales
- Apply meaningful colors
- Present complete information
- Use appropriate chart types
